# Covid Patient Timeline

## Prerequisite
* Docker [How to install](https://docs.docker.com/engine/install/)

---
## Steps
1. Edit script's permission
  ```sh
    chmod +x ./start.sh
  ```
2. Build and start docker containers
  ```sh
    ./start.sh
  ```
4. Enter http://0.0.0.0:3000 in your browser
